# PyTitration-Sim-Lab 🧪

### ## 主題 (Subject)
**Interactive Virtual Acid-Base Titration Laboratory**

### ## Description
This project is an educational simulation tool built with Python and Pygame. it allows users to perform acid-base titration experiments in a risk-free virtual environment. The application calculates chemical equilibrium in real-time and visualizes the process through dynamic liquid color transitions and live titration curve plotting.

**Key Features:**
* **Chemical Versatility:** Supports combinations of Strong/Weak acids (HCl, CH3COOH) and Strong/Weak bases (NaOH, NH4OH).
* **Indicator System:** Accurate color logic for Methyl Orange, Phenolphthalein, Bromothymol Blue, and more.
* **Real-time Analytics:** Instant pH calculation and dynamic titration curve graphing.
* **Data-Driven:** Easily update chemical properties via the linked `data.xlsx`.
